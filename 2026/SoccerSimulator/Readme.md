# SoccerSimulator

Simulador de gerenciamento de futebol desenvolvido em Java com foco na aplicação dos conceitos de Programação Orientada a Objetos.

## Sobre o Projeto

SoccerSimulator é um jogo de linha de comando (CLI) onde o jogador assume o controle de um clube de futebol e deve tomar decisões administrativas e esportivas para desenvolver sua equipe ao longo das temporadas.

O projeto foi criado com o objetivo principal de praticar os quatro pilares da Programação Orientada a Objetos (POO), aplicando conceitos como abstração, encapsulamento, herança e polimorfismo em um cenário de gerenciamento esportivo.

## Funcionalidades

* Gerenciamento financeiro do clube
* Contratação e demissão de presidentes
* Contratação e demissão de treinadores
* Contratação e venda de jogadores
* Investimentos na estrutura do clube
* Simulação de temporadas
* Evolução do clube ao longo do jogo

## Conceitos Aplicados

### Encapsulamento

Controle e proteção dos dados internos das entidades do sistema.

### Herança

Reutilização de código através da especialização de classes.

### Polimorfismo

Implementação de diferentes comportamentos por meio de interfaces e hierarquias de classes.

### Abstração

Representação dos elementos do mundo do futebol através de modelos orientados a objetos.
